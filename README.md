# Shared Skills

A collection of reusable Claude Code skills.

## How to use

Don't read through these manually — just point your LLM at this repo and let it figure out what's available and how to use them.

For example, tell Claude:

> "Check the skills in https://github.com/leo-li-opus/shared_skills and install any that are relevant."

Claude will read the skills, understand what they do, and set them up for you.
